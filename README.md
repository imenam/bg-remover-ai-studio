# 🎯 BG Remover AI Studio

A full-stack AI-powered background removal web application built with **FastAPI** and **rembg (ISNet model)**, featuring a custom JavaScript canvas crop engine and dynamic background composition system.

---

## 🚀 Features

### 1️⃣ Smart Background Removal
- Powered by `rembg` with ISNet model
- High-quality PNG output with transparent background
- 20MB file size validation
- JPEG / PNG / WEBP support

### 2️⃣ Advanced Crop Engine (Custom Built)
- Interactive canvas-based crop system
- Drag, resize, move crop selection
- Lockable aspect ratios (1:1, 16:9, 4:3, custom)
- Manual pixel input controls
- Real-time preview

### 3️⃣ Background Composer
- Solid color backgrounds
- Gradient backgrounds (direction control)
- Transparent preview mode
- Preset swatches + custom color picker

### 4️⃣ Export Options
- PNG (transparent preserved)
- JPG (with selected background)
- WEBP (optimized size)
